App Demo que implementa en un test el ejemplo el Quick Start del tool Mongo-Hadoop con Spring y utilizando un sistema Hadoop remoto pseudodistribuido.

He modificado ligeramente el ejemplo para:

  • Tomar datos de MongoDB, procesar los resultados en Hadoop y devolver los resultados a MongoDB.
  • Tomar los datos del HDFS de Hadoop y devolver los resultados a MongoDB.

Requisitos:

  • Seguir la guía de Quick Start MongoDB para obtener el juego de datos de prueba: http://www.mongodb.org/display/DOCS/Hadoop+Quick+Start#HadoopQuickStart-TreasuryYield

  • Una base de datos MongoDB

  • Un sistema Hadoop instalado en modo Pseudodistribuido.

  • Instalar la dependencia maven mongo-hadoop que encontrarás ya compilada en /src/main/lib/
